*Cassette Sales Page*
Cassette sales page is basic sales page serving the purpose of giving in stock items from a database to the user and allowing them to add and remove them from a cart. 

#### Technologies
HTML/CSS, Javascript, Axios, Nodemon, Local Database

#### Features
1. Uses a database to supply in stock items to the user of the page
2. Images and information about the in stock items are supplied from the database
3. User is able to add items to a cart
4. User is able to remove items from the cart
